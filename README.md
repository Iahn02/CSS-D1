# CSS-D1
Desafío 1 - CSS avanzado

Este proyecto es parte del bootcamp "Desafío Latam". En este desafío, se hace un uso completo y eficiente de Flexbox con CSS para crear un diseño responsivo y moderno. El objetivo es dominar las técnicas avanzadas de CSS y aplicarlas en un entorno práctico y profesional.
